Ayuda: Visualizar Agenda

Los siguientes pasos le guiarán a través del proceso de viasualización de la agenda de un doctor:

  • ID Cita: Muestra el id de la cita.
  • ID Paciente: Muestra el id del paciente asociado a la cita.
  • ID Doctor: Muestra el id del doctor de la cita.
  • ID Tratamiento: Muestra el id del tratamiento realizado
  • Fecha Hora: Muestra la fecha y hora de la cita.
  • Motivo visita: Muestra el motivo de la visita.

Si tiene alguna pregunta o necesita ayuda adicional, no dude en comunicarse con nosotros.
